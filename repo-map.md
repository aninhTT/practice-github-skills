# Repo Map

Use this file as a quick guide when exploring the repo with Codex or Claude Code.

## Root Files

- `README.md`: Main learner guide and skill rules.
- `CONTRIBUTING.md`: Procedural contribution instructions.
- `AGENTS.md`: Instructions for Codex and other coding agents.
- `CLAUDE.md`: Instructions for Claude Code.
- `repo-map.md`: This orientation map.

## Folders

- `skills/`: Fake skills used for the practice exercise.
- `.github/ISSUE_TEMPLATE/`: GitHub issue forms.
- `.github/workflows/`: GitHub Actions validation.

## Expected Learner Change

Learners should add exactly one new file:

```text
skills/<first-name>-<skill-name>/SKILL.md
```
