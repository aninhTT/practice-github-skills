# Wyatt Board Game Rules Summarizer

## What It Does

Summarizes the rules for a fictional board game into a friendly quick-start guide, including the goal, setup, turn flow, scoring, and a short example round.

## When To Use It

Use this skill when practicing how to turn made-up game rules into a clear summary for new players.

## Inputs

- A fictional board game name.
- A made-up player count and play time.
- Invented setup details.
- Pretend turn rules.
- Fictional scoring or win conditions.

## Output

A concise rules summary with sections for game goal, setup, turn order, scoring, and one fictional example turn.

## Example Prompt

```text
Summarize the rules for a fictional board game called Moon Market, where players trade imaginary moon gems to build the best space bazaar.
```

## Safety Notes

Use only fictional game names, rules, examples, and scenarios. Do not copy real board game rulebooks, private playtest notes, company material, customer data, credentials, or personal information.
