# Amie Spreadsheet Cleaner

## What It Does

Helps turn a messy fictional spreadsheet into a cleaner table by suggesting consistent column names, removing duplicate rows, and flagging missing values.

## When To Use It

Use this skill when a learner wants to practice describing a simple data cleanup workflow without using real business data.

## Inputs

- A fictional description of the spreadsheet.
- The column names.
- The cleanup goal.
- Any fake sample rows the learner wants to include.

## Output

A short cleanup plan, suggested column names, and a checklist of safe next steps.

## Example Prompt

```text
I have a fake spreadsheet of community garden signups with inconsistent column names and duplicate rows. Help me make a cleanup plan.
```

## Safety Notes

Use invented spreadsheet examples only. Do not use real company spreadsheets, customer lists, employee data, personal files, or private records.
