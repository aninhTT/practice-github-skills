# Amie Repo Tour Guide

## What It Does

Helps a learner explore a fictional practice repository by summarizing the main files, explaining where contributions belong, and suggesting one safe next step.

## When To Use It

Use this skill when someone opens a practice repo for the first time and wants a friendly orientation before making a branch or editing files.

## Inputs

- A fictional repository goal.
- A short list of folders or files to explain.
- The learner's practice objective.
- Any public-safe questions about the repo structure.

## Output

A short repo tour, a list of important files, and one suggested fictional contribution idea.

## Example Prompt

```text
Give me a tour of this fake GitHub practice repo and suggest one public-safe skill I could add.
```

## Safety Notes

Use only the files in the practice repo and invented examples. Do not use personal history, private files, company information, customer data, credentials, or real workflows.
